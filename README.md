## HTML

- HTML - Hypertext Markup Language
- HTML is the standard markup language for creating web pages and web applications.
- Responsible for webpage structure
- <element>content</element>
- <h1>Heading</h1>
- <p>Paragraph</p>
- <img src=''> === image element
- <a href=''>Link</a>
- <br/> === line break
